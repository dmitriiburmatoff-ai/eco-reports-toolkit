# eco-reports-toolkit
eco-reports-toolkit — скрипты для автоматизации подготовки природоохранной документации (инвентаризация, НДВ, НМУ, СЗЗ, шум)
Этот проект создан для внутреннего использования и предоставляется «как есть». Он не является официальным продуктом и не связан с разработчиками коммерческого ПО для экологического моделирования.
